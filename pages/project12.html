Project Details :

  • Name : Weather App
  • About : The Interactive Weather Application is a dynamic web app developed with ReactJS, providing real-time weather information using the OpenWeatherMap API. Users can search for any city to obtain detailed weather data, including temperature, wind speed, and humidity. The app features robust error handling for invalid city searches and ensures a smooth, responsive user experience. This project highlights my proficiency in creating interactive, data-driven web applications with modern JavaScript frameworks and APIs.
